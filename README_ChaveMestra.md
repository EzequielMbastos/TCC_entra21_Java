ChaveMestra

Sistema de gestão para comércios com assistente de consulta.
Cliente real: Loja de Chaves - Itajaí/SC.

Problema

Comerciantes perdem horas digitando notas fiscais, não têm visibilidade
rápida sobre estoque e contas, e sofrem com sistemas caros ou que param
sem internet. O mercado vende IA como solução mágica, mas entrega
caixas-pretas que tomam decisões sozinhas. O ChaveMestra oferece um
assistente com poder apenas de consulta, que lê notas e responde
perguntas. Sem internet, o sistema continua funcionando.

Funcionalidades do MVP

1. Cadastro de produtos
2. Entrada de estoque via upload de nota fiscal
3. Registro de vendas com baixa automática
4. Contas a pagar com alerta de vencimento
5. Dashboard com visão geral do negócio
6. Leitura automática de nota fiscal
7. Assistente de consulta por texto ou voz

Entidades

Produto, Categoria, Fornecedor, EntradaEstoque, Venda, ContaPagar
